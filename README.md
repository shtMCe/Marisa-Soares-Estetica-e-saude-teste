---

## 🤖 Abordagem de Desenvolvimento: IA Generativa + Engenharia de Software

**Nota para Recrutadores:** Este projeto foi concebido sob uma ótica moderna de produtividade e eficiência tecnológica.

A estrutura inicial e parte dos blocos de código foram gerados utilizando ferramentas de **Inteligência Artificial (IA)**. O meu papel principal neste projeto consiste na **curadoria, arquitetura, revisão de código (*Code Review*), refatoração e adaptação** para o cenário real e para as regras de negócio da cliente.

### Por que escolhi esta abordagem?
* **Aumento de Produtividade:** Utilizo a IA como um acelerador de *boilerplate* (estruturas de código repetitivas), permitindo-me focar na experiência do usuário, no refinamento visual e na lógica de negócio.
* **Engenharia de Prompt & Filtro Crítico:** Gerar código com IA exige saber o que pedir e, mais importante, saber validar se o código entregue é seguro, semântico e performático. Estou aprendendo a usar a IA para facilitar o meu dia a dia, mantendo o controle total sobre a qualidade final.
* **Capacidade de Adaptação:** O código gerado por ferramentas automatizadas raramente vem pronto para produção. Ler, refatorar e adaptar essa base demonstra a minha aptidão em compreender e melhorar códigos de terceiros.
